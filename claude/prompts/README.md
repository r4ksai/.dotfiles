# Prompts

Standalone prompt text for recurring one-off tasks that don't warrant a full
skill or subagent — e.g. a code review checklist, a commit-message style guide,
a PR-description template.

## Adding a new prompt

1. Name the file after its purpose, e.g. `code-review-checklist.md`,
   `commit-message-style.md`.
2. Keep each file focused on a single task.
